A simple, clean CV helps fresh candidates get shortlisted faster.

## CV Sections You Need

- Full name and contact number
- Professional email address
- Education details
- Skills section
- Short objective statement

## Keep It Simple

- Use one page
- Avoid unnecessary graphics
- Use clear headings
- Check spelling and phone number

## Common Mistakes

- Adding fake experience
- Using unclear email IDs
- Sending CV without renaming file

Use a file name like `Ali-Khan-CV.pdf`.
